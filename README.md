# How_To_Flash_Dual_C5


This repository contains a PDF guide explaining how to flash Marauder firmware to the AWOK Dynamics C5 Dual Touch.

## Download Guide

- [Download PDF Guide](Dual_C5_Touch_Flash_Process.pdf)

This guide depends on files and tools from these projects:

- [AWOK559 / C5_Py_Flasher](https://github.com/AWOK559/C5_Py_Flasher)

- [justcallmekoko / ESP32Marauder](https://github.com/justcallmekoko/ESP32Marauder)
